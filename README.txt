------------------ RUNNING CODE ------------------
1. If downloading data directly instead of using with source code, it is necessary to put both data sets in the data directory and run the combine_data.py script to combine into one uniform dataset

2. Run initial experiments using `python run_experiment.py --all`
3. Plot the results so far using `python run_experiment.py --plot`
4. The dimension values in run_clustering.sh have been updated using the values found from the plots
5. Run run_clustering.sh
6. Output final charts using `python run_experiment.py --plot`
